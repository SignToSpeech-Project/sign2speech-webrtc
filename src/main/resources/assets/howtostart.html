Sign2Speech is very easy to use. Just follow the steps below to get started:

STEP 1:
Connect to the webchat by selecting a username and a room name. If you want to join a friend who is already connected in a room, ask your friend the name of the room

STEP 2:
If you're the one using the camera, launch the Sign2Speech application using Visual Studio and select the "Recognition mode"

STEP 3:
Enjoy! :)


If you are having some troubles using Sign2Speech, make sure you are connected to internet and use Mozilla Firefox.
